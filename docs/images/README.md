# Reference Figures

Diagnostic and reference figures generated during pipeline development and
simulation audit. All figures use the SciencePlots `science + notebook` style.

## Array Geometry

| File | Description | Generated by |
|---|---|---|
| `dsa110_antenna_layout.png` | ENU layout of all 117 station positions. Core T-shape (EW arm + NS arm) clearly visible, with 14+ outriggers extending to ~1.8 km E-W and ~2.2 km N-S. | `scripts/plot_tile_image.py` |
| `dsa110_psf_analysis.png` | Synthesized beam analysis at Dec +16.15°, 1405 MHz. 6-panel: UV coverage, PSF wide view, main beam zoom, E-W slice, N-S slice, and parameter summary. Shows how T-core vs outrigger baselines shape the dirty beam. | `scripts/make_psf_analysis.py` |

**Key beam parameters (117 antennas, Dec +16.15°, 1405 MHz):**

| Parameter | Value |
|---|---|
| HPBW E-W (full array) | 75" |
| HPBW N-S (full array) | 332" (4.4× elongated) |
| Peak sidelobe | 0.415 |
| UV fill fraction | 0.90% |
| u range | ±48.9 kλ |
| v range | ±10.9 kλ |

| `step6_check3_noise_decomposition.png` | rms vs. coverage depth — noise decomposition into thermal and incoherent sidelobe components, with shared-sky analytical prediction. | Step 6 |
